## 5トーン (fiveToneScale)

### 概要
- **構成音**: 5種類
- **音域幅**: 7半音（完全5度）
- **用途**: 基本的な発声練習

### インターバル
```
[0, 2, 4, 5, 7]
```

| 音程 | 半音数 | 間隔 |
|------|--------|------|
| ド | 0 | - |
| レ | 2 | 2半音（長2度） |
| ミ | 4 | 2半音（長2度） |
| ファ | 5 | 1半音（短2度） |
| ソ | 7 | 2半音（長2度） |

### 演奏パターン (ascendingDescending)
```
上昇: ド → レ → ミ → ファ → ソ
下降: ファ → ミ → レ → ド

[0, 2, 4, 5, 7, 5, 4, 2, 0]
```

**演奏音数**: 9音

---

## オクターブリピート (octaveRepeat)

### 概要
- **構成音**: 4種類
- **音域幅**: 12半音（1オクターブ）
- **用途**: 音域拡張、オクターブ跳躍練習
- **特徴**: トップ音を4回繰り返す

### インターバル
```
[0, 4, 7, 12]
```

| 音程 | 半音数 | 間隔 |
|------|--------|------|
| ド | 0 | - |
| ミ | 4 | 4半音（長3度） |
| ソ | 7 | 3半音（短3度） |
| ド(高) | 12 | 5半音（完全4度） |

**特徴**: メジャートライアド + オクターブ構造

### 演奏パターン
```
上昇: ド → ミ → ソ → ド(高)
トップ4回: ド(高) → ド(高) → ド(高) → ド(高)
下降: ソ → ミ → ド

[0, 4, 7, 12, 12, 12, 12, 7, 4, 0]
```

**演奏音数**: 10音

### 実装上の注意
- 標準の `ascendingDescending()` では対応できない
- トップ4回リピートの専用ロジックが必要
